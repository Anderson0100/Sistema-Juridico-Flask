# Sistema Jurídico

Sistema de gestão para escritórios de advocacia feito em Flask + MySQL.

## Funcionalidades
- Login (Admin, Advogado, Cliente)
- Processos
- Prazos
- Audiências
- Upload de PDFs
- Google Calendar
- Notificações
- Segurança de acesso

## Como rodar
pip install -r requirements.txt
python app.py

## Variáveis de ambiente
Crie um arquivo .env com:
SECRET_KEY
DATABASE_URL
GOOGLE_CLIENT_ID
GOOGLE_CLIENT_SECRET
